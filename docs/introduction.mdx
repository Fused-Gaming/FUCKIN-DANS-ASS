# Blockchain Forensic Analysis Toolkit

**Professional-grade on-chain investigation framework with AI-powered MCP integration and Etherscan API V2 support**

## Overview

The Blockchain Forensic Analysis Toolkit is a comprehensive, open-source forensic toolkit designed for investigating blockchain-based criminal activity, including illegal gambling operations, fraud networks, money laundering, and asset theft. Built with a focus on **evidence quality**, **court admissibility**, and **investigative efficiency**.

## Why This Toolkit?

Traditional blockchain explorers provide raw data. This toolkit provides **actionable intelligence**:
